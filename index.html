The ApoZomb Installer

El instalador automáticamente descargará e instalará ApoZomb + Shaders con el programa Iris Shaders para poder cargarlos sin problemas dentro del juego. Inicia el instalador, elige tu estilo de shaders y presiona instalar. Java es requerido para iniciar el instalador. (El Java de minecraft no será funcional en este caso)
